# 해시 테이블

> 해시테이블은 key값을 받아서 value값을 반환하는 함수입니다. python에서는 Dictionary를 의미합니다. 해시함수와 배열을 결합해서 만듭니다.

``` python
price = {'avocado':3500,'apple':800}
           [key]  [value]
```



* 하나의 key에 대해서  여러 value를 할당할 수 없습니다.
* 유효한 key값만 반환할 수 있습니다.



## (1) 해시테이블의 장점

* 어떤것과 다른것 사이의 관계를 모형화 할 수 있습니다.
* 중복을 막을 수 있습니다.
* 서버에게 작업을 시키지 않고 자료를 캐싱할 수 있습니다.



## (2) 해시테이블의 성능

|      | 일반적상황 | 충돌상황 |
| ---- | ----- | ---- |
| 탐색   | O(1)  | O(n) |
| 삽입   | O(1)  | O(n) |
| 삭제   | O(1)  | O(n) |

* 좋은 해시 테이블은 충돌 상황을 피해야합니다. 충돌을 피하기 위해서 **낮은 사용률**(리사이징), **좋은 해시함수**(배열에 값을 고르게 분포시키는 함수)가 필요합니다.
